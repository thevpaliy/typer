# typing-test

# Running locally:

1. Get the project on your machine.

```git clone https://github.com/thevpaliy/typing-test && cd typing-test```

2. Run an HTTP local server with Python:.

```python -m SimpleHTTPServer``` 

   Or if you prefer Python 3, then you can utilize this:
   
```python -m http.server``` Python 3 only

3. Open http://localhost:8000
